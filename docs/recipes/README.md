# Recipes

Short, task-focused how-to guides for common SEJA workflows. Each recipe covers one specific task with prerequisites, step-by-step instructions, and tips.

## Getting Started

| Recipe | What it covers |
|--------|---------------|
| [Bootstrap a Greenfield Project](recipe-bootstrap-greenfield.md) | Create a new project from scratch with `/seed` and `/design`. |
| [Map an Existing Codebase](recipe-map-existing-codebase.md) | Discover and document an existing system using `/explain` and `/advise --inventory`. |
| [Set Up a Project Workspace](recipe-workspace-setup.md) | Create a separate workspace for framework artifacts alongside an existing codebase. |

## How to use recipes

Recipes are designed to be self-contained -- you can follow any recipe without reading the others. If you are new to SEJA, start with the [bootstrap recipe](recipe-bootstrap-greenfield.md) or pick a [journey](../journeys/) that matches your situation for a broader workflow overview.
